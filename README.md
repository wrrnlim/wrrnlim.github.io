## Development

### Install Ruby

```shell
sudo apt install ruby-full
```

### Install [Jekyll](https://jekyllrb.com/docs/installation/) and Bundler

```shell
sudo gem install jekyll bundler
```

To build locally, install dependencies:

```shell
bundle install
```

and run

```shell
bundle exec jekyll serve
```
